新增的数据展示 Previously Hidden

以下数据 main process 已经捕获并存储，但当前 UI 没有展示。在新设计中通过 Inspector Overview 和 conversation 增强来展示。

Inspector Overview 新增

  • Token Usage — inputTokens, outputTokens, reasoningTokens (卡片式)
  • Stop Reason — end_turn, max_tokens, tool_use 等
  • Request/Response Size — 格式化显示 (KB)
  • Profile Name — 来自哪个 profile 的捕获
  • Endpoint Kind — messages / responses

Conversation 增强

  • Tool callId 关联 — tool-call 和 tool-result 通过 callId 配对显示
  • isError 标记 — tool-result 的 isError 用红色状态标记
  • Reasoning 耗时 — 折叠标题显示 thinking 时长和 token 数
  • Instructions 预览 — system 消息在 conversation 顶部显示

Settings 增强

  • 端口号 — 每个 profile 显示监听端口
  • 运行状态开关 — 直接在 settings 里启停 profile
  • enabled / autoStart — 对应 ConnectionProfile 的两个字段
